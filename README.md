 # AQI Forecasting bhopal
 
 -> About the Project

This project studies how environmental conditions change over time and different locations. It focuses on analyzing data such as temperature, pollution, or weather patterns to understand trends and patterns. The project uses basic concepts of AI and Machine Learning to process data and present meaningful insights.

__________________________________________________________

 -> Aim

To understand environmental changes by analyzing data across time and space.
